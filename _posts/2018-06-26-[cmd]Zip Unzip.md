- 壓縮

``` python
zip -r xxx.zip /.../xxx  
```

- 解壓縮

``` python
unzip xxx.zip
```

``` python
unrar -e xxx.rar
```

- 隱藏檔案

``` python
defaults write com.apple.finder AppleShowAllFiles TRUE
killall Finder
```

- 還原隱藏

``` python
defaults write com.apple.finder AppleShowAllFiles FALSE
killall Finder
```

- 批量轉副檔名
-- JAVA轉成TXT

``` python
find . -iname "*.java" -exec bash -c 'mv "$0" "${0%\.java}.txt"' {} \;
```

- 開啟第二個eclipse

``` python
open -n /Applications/Eclipse.app
```
